Создать систему управления учениками курсов.
Необходимо реализовать возможность создавать / редактировать / удалять курс.
У курса есть два поля:
1. Название курса (длина текста не более 128 символов )
2. Преподаватель (длина текста не более 128 символов )
На курс необходимо иметь возможность добавить / редактировать / удалить ученика.
У ученика есть следующие пола:
1.
2.
3.
4.
Фамилия (длина текста не более 128 символов )
Имя (длина текста не более 128 символов )
Email (длина текста не более 128 символов, корректный email )
Фотография
Список курсов и учеников должны выводится в виде сводной таблицы.
Все данные введенные пользователем сохраняются в БД MySQL
Система должна корректно работать на следующей конфигурации:
PHP 7.2+
MySQL 5.7+
Apache 2.4+
Выложить файлы используемые в задании в виде архива, любым удобным
способом(например загрузить на файлообменик, как приложение к письму или выложить
проект на git).
В файлы проекта необходимо приложить результирующую базу данных с несколькими
заполненными курсами и учениками
